# Boltzmann Wealth Model
## What this model does
This model simulates wealth exchange between agents. Each agent starts with 1 unit of wealth. At each step, agents randomly give 1 unit of wealth to another agent.
## What I learned
-Basic idea about Mesa
- How Mesa agents and models work
- How emergent behavior appears from simple rules
- Even with fair rules, wealth inequality emerges naturally
## Mesa features used
- mesa.Agent
- mesa.Model
- agents.shuffle_do()
- DataCollector

I am new to concepts of ABM.The concept of ABM felt intresting.And coming to this model,everyone starts equal but wealth inequality still emerges. This shows how real world inequality can appear from simple interactions.
